## Установка

#### Устанвите Python верссии 3.7 или выше.
#### Добавьте Python в PATH
#### Склонируйте этот репозиторий
git clone https://github.com/unbrokenguy/infosearch.git
#### Перейдите в папку с репозиторием и установите скрипт
```shell
cd crawler
pip install -e .
```
## Использование
```shell
crawler -url <url>
```
